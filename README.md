# 💡 Animated Lamp Login Page

A modern, responsive, and interactive **Animated Lamp Login Page** built using **HTML, CSS, and JavaScript**. This project was developed as a **front-end practical project** to enhance my web development skills and explore CSS animations, responsive layouts, and JavaScript interactions.

---

## 📖 About the Project

The **Animated Lamp Login Page** is a creative login interface featuring a clickable hanging lamp that adds an engaging visual experience. Along with its modern UI, the project includes essential login components such as email and password fields, password visibility toggle, social login buttons, and a responsive layout.

This project helped me gain practical experience in building interactive and user-friendly web pages using core front-end technologies.

---

## ✨ Features

- 💡 Interactive animated hanging lamp
- 🎨 Modern and clean user interface
- 📱 Fully responsive design
- 🔐 Email and password login form
- 👁️ Show/Hide password functionality
- ✅ Remember Me option
- 🔗 Forgot Password link
- 🌐 Google and GitHub login buttons (UI)
- ⚡ Smooth animations and transitions
- 🎯 Beginner-friendly project structure
